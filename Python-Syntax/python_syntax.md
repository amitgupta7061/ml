# Python Syntax Basics

Python is a high-level, interpreted programming language known for its simplicity and readability.

## 1. Variables and Data Types
Variables are containers for storing data values. Python has no command for declaring a variable; it is created the moment you first assign a value to it.
Common data types:
* **Integers (`int`)**: Whole numbers (positive or negative).
* **Floats (`float`)**: Numbers with a decimal point.
* **Strings (`str`)**: Text, enclosed in single or double quotes.
* **Booleans (`bool`)**: True or False values.

## 2. Conditionals (`if`, `elif`, `else`)
Conditional statements are used to perform different actions based on different conditions. In Python, indentation is used to define the scope of the block.

## 3. Loops (`for`, `while`)
* **`for` loop**: Used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).
* **`while` loop**: Executes a set of statements as long as a condition is true.
* Control statements: `break` (exits the loop), `continue` (skips the current iteration), `pass` (does nothing, placeholder).

## 4. Data Structures
* **Lists (`list`)**: Ordered, mutable (changeable), allows duplicate members. Written with square brackets `[]`.
* **Tuples (`tuple`)**: Ordered, immutable (unchangeable), allows duplicate members. Written with round brackets `()`.
* **Sets (`set`)**: Unordered, unindexed, no duplicate members. Written with curly brackets `{}`.
* **Dictionaries (`dict`)**: Unordered, mutable, indexed by keys. Written with curly brackets `{}` with key-value pairs.

## 5. Functions
A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. Functions are defined using the `def` keyword.
